#### Setting ResultSet locally

```shell
mongod --dbpath /data/rs1 --replSet rs0 --port 27017
```

Explanation:

```
--dbpath → data directory (must exist)

--replSet rs0 → replica set name

--port 27017 → default MongoDB port
```

If /data/rs1 doesn’t exist:

```shell
mkdir -p /data/rs1
```
